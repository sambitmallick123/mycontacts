# py-contacts
Python program (mac OS) for adding contact details, showing contact details to/from a YAML file

USAGE :

Help :
python py-contacts.py -h
python py-contacts.py --help

Version :
python py-contacts.py --version

Add Contact Details :
python py-contacts.py -N <enter name> -E <enter email id> -M <enter mobile number>
e.g.:
py-contacts.py -N Sambit -E sambit@gmail.com -M 9988765342

Show Contact Details :
python py-contacts.py -S <enter name>
e.g.:
python py-contacts.py -S Sambit

Display complete contact list :
python py-contacts.py -L all
